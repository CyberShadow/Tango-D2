Install Tango:

Windows:
========

1. Ensure you have the 'Digitalmars' dmd compiler version 0.163 or greater installed.
2. Be sure that the 'Digitalmars' make is the first make command in your path.
3. Open a cmd shell in the lib directory.
4. start build-dmd.bat

Linux with DMD:
===============

1. Ensure you have the 'GNU' make version 3.79 (???) or greater installed.
2. Ensure you have the 'Digitalmars' dmd compiler version 0.163 or greater installed.
3. Open a shell in the lib directory.
4. Start ./build-dmd.sh
