    • A bridge between a Format instance and a Buffer. This is used for the Stdout & Stderr globals, but can be used for general-purpose buffer-formatting as desired. The Template type 'T' dictates the text arrangement within the target buffer ~ one of char, wchar or dchar (utf8, utf16, or utf32)

    • Construct a BufferFormat instance, tying the provided buffer to a formatter. Set option 'flush' to true if the result should be flushed when complete, and provide a 'tail' to append on the completed output (such as a newline).

  • static BufferFormatT Stdout ;
    • Standard, global formatters for console output. If you don't need formatted output or unicode translation, consider using the module io.Console directly
