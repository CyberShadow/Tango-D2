  • class FilePath ;
    • Models a file name. These are expected to be used as the constructor argument to File implementations. The intention is that they easily convert to other representations such as absolute, canonical, or Url. Note that this class is immutable. Use MutableFilePath if you wish to alter specific attributes.

    File paths containing non-ansi characters should be UTF-8 encoded. Supporting Unicode in this manner was deemed to be more suitable than providing a wchar version of FilePath , and is both consistent & compatible with the approach taken with the Uri class.

  • this();
    • Create an empty FilePath. This is strictly for subclass use.

  • this(FilePath other);
    • Create a FilePath through reference to another.

  • this(char[] filepath, bool copy = true);
    • Create a FilePath from the given string. Note the path string is usually duplicated here, though you may specify that it be aliased instead via the second argument. When aliased, you are expected to provide an immutable copy for the lifetime of this object. If you are not certain, ignore the second argument.

  • static char[] normalize (char[] path);
    • Convert path separators to the correct format. This mutates the provided 'path' content, so .dup it as necessary.

  • static char[] replace (char[] path, char from, char to);
    • Replace all 'from' instances in the provided path with 'to'

  • protected void reset ();
    • Clear any cached information in this FilePath

  • bool isAbsolute ();
    • Returns true if this FilePath is *not* relative to the current working directory.

  • bool isEmpty ();
    • Returns true if this FilePath is empty

  • char[] getRoot ();
    • Return the root of this path. Roots are constructs such as "c:".

  • char[] getPath ();
    • Return the file path. Paths start with a '/' but do not end with one. The root path is empty. Directory paths are split such that the directory name is placed into the 'name' member.

  • char[] getName ();
    • Return the name of this file, or directory.

  • char[] getExtension ();
    • Return the file-extension, sans seperator

  • char[] getSuffix ();
    • Suffix is like extension, except it can include multiple '.' sequences. For example, "wumpus1.foo.bar" has suffix "foo.bar" and extension "bar".

  • void delegate(void[]) produce (void delegate(void[]) consume);
    • Convert this FilePath to a char[] via the provided Consumer

  • char[] toString ();
    • Convert this FilePath to a char[]

  • char[] toUtf8 (bool withNull = false);
    • Return a zero terminated UTF8 version of this file path

  • wchar[] toUtf16 (bool withNull = false);
    • Return a zero terminated UTF16 version of this file path

  • char[] splice (FilePath base);
    • Splice this FilePath onto the end of the provided base path. Output is return as a char[].

  • void delegate(void[]) splice (FilePath base, void delegate(void[]) consume);
    • Splice this FilePath onto the end of the provided base path. Output is handled via the provided Consumer

  • FilePath toParent ();
    • Returns a FilePath representing the parent of this one. An exception is thrown if there is not parent (at the root).

  • bool isChild ();
    • Returns true if this FilePath has a parent.

  • FilePath toSibling (char[] name);
    • Return a cloned FilePath with a different name.

  • FilePath toSibling (char[] name, char[] ext);
    • Return a cloned FilePath with a different name and extension. Note that the suffix is destroyed.

  • FilePath toSibling (char[] name, char[] ext, char[] suffix);
    • Return a cloned FilePath with a different name, extension, and suffix.

  • class MutableFilePath : tango.io.FilePath.FilePath;
    • Mutable version of FilePath, which allows one to change individual attributes. A change to any attribute will cause method toString() to rebuild the output.

  • this();
    • Create an empty MutableFilePath

  • this(FilePath other);
    • Create a MutableFilePath through reference to another.

  • this(char[] name);
    • Create a MutableFilePath via a filename.

  • MutableFilePath setExt (char[] ext);
    • Set the extension of this FilePath.

  • MutableFilePath setName (char[] name);
    • Set the name of this FilePath.

  • MutableFilePath setPath (char[] path);
    • Set the path of this FilePath.

  • MutableFilePath setRoot (char[] root);
    • Set the root of this FilePath (such as "c:")

  • MutableFilePath setSuffix (char[] suffix);
    • Set the suffix of this FilePath.
