  • enum Interval ;
    • Time interval multipliers ~ all Tango intervals are based upon microseconds. Note that recent D compilers support arithmetic applied upon Interval members to be passed as a fully-typed Interval argument; e.g. 

            void sleep (Interval period);

        sleep (Inteval.second * 5);
    Intervals can currently extend up to a one-hour period. They're intended to represent short durations of time, such as when one waits for a socket response.
