  • class IOException : object.Exception;
    • The basic exception thrown by the tango.io package. One should try to ensure that all Mango exceptions related to IO are derived from this one.

  • this(char[] msg);
    • Construct exception with the provided text string

  • class EofException : tango.io.Exception.IOException;
    • This exception is thrown by Readers whenever the file content is terminated unexpectedly. A Writer throws the exception when it fails to append data to an externl conduit.

  • this(char[] msg);
    • Construct exception with the provided text string

  • class TokenException : tango.io.Exception.IOException;
    • These exceptions are thrown by the Token subsystem, typically for invalid data formatting or other content related issues.

  • this(char[] msg);
    • Construct exception with the provided text string

  • class PickleException : tango.io.Exception.IOException;
    • PickleException is thrown when the PickleRegistry encounters a problem during proxy registration, or when it sees an unregistered guid.
