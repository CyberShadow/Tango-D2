This module implements the MD5 Message Digest Algorithm as described by RFC 1321 The MD5 Message-Digest Algorithm. R. Rivest. April 1992.

  • class Md5Digest : tango.math.cipher.Md4.Md4Digest;


  • this(uint[4] raw);
    • Construct an Md5Digest.

    Remarks:
    Constructs an Md5Digest from binary data



  • class Md5Cipher : tango.math.cipher.Md4.Md4Cipher;


  • Md5Digest getDigest ();
    • Obtain the digest

    Returns:
    the digest

    Remarks:
    Returns a digest of the current cipher state, this may be the final digest, or a digest of the state between calls to update()



  • protected void transform (ubyte[] input);
    • Performs the cipher on a block of data

    Params:
    data the block of data to cipher

    Remarks:
    The actual cipher algorithm is carried out by this method on the passed block of data. This method is called for every blockSize() bytes of input data and once more with the remaining data padded to blockSize().
