  • static BufferedFormat Stdout ;
    • Standard, global formatters for console output. If you don't need formatted output or unicode translation, consider using the module tango.io.Console directly

    Note that both the buffer and conduit in use are exposed by these global instances ~ this can be leveraged, for instance, to copy a file to the standard output: 

            Stdout.conduit.copy (new FileConduit ("myfile"));
