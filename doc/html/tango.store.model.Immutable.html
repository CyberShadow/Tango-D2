• interface Immutable ;
    • Immutable is a root interface of immutable classes; i.e., those with objects that may be looked at but not updated. By necessity in Java, all implementation classes supporting interface Immutable must be final classes.

    There are no operations defined in this interface. This interface is used only to provide a name for this `Purity' property.
